**sort output list**
```
psort.py -z utc -o list
```

**sort plaso dump**
```
psort.py -z utc -o l2tcsv -w plaso_timeline.csv /path/to/casename.dump
```

**sort plaso dump with timeframe**
this shows 5 mins before and 5 mins after the sliced time
```
psort.py -z utc -o l2tcsv -w plaso_timeline.csv /path/to/casename.dump --slice `2026-06-08 15:00`
```

**sort plaso dump with date range**
this shows events from June 1 12am to June 8 12am 
```
psort.py -z utc -o l2tcsv -w plaso_timeline.csv /path/to/casename.dump "date > `2026-05-31 23:59:59` AND date < `2026-06-09 00:00:00`"
```
