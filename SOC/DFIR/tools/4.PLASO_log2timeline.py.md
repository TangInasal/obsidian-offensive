### **BASIC USAGE (kitchen sink)**

```plaso
log2timeline.py casename.dump "C:\Users\User\DFIR\cases\KAPE_Triage_Image"
```

### PARSER

**parser list**
```
log2timeline.py --parsers list | more
```

**parser usage sample (webhistory)**
```
log2timeline.py --parsers webhist
```

**exclude parser (winevt)** 
```
log2timeline.py --parsers webhist,-winevt
```
OR
```
log2timeline.py --parsers webhist,\!winevt
```

### FILTER
[plaso filter link](https://github.com/mark-hallman/plaso_filters/blob/master/filter_windows.txt)

**with filter file**
```
log2timeline.py -f /path/to/filter_windows.txt casename.dump /path/to/triage_image
```

**with filter and parser (webhist)**
```
log2timeline.py --parsers webhist -f /path/to/filter_windows.txt casename.dump /path/to/triage_image
```
